# Le Marché Rural

Thème WordPress marketplace agricole locale, pensé pour WooCommerce + WCFM, avec une base Gutenberg maintenue légère et performante.

## Version en cours

**2.5.0 — Marketplace Theme Overhaul**

### Objectifs livrés

- design system consolidé (variables, cartes, badges, sections, boutons)
- header réorganisé (logo, navigation primary, actions panier)
- footer structuré (menu secondaire + menu légal + copyright)
- amélioration visuelle de la boutique et des fiches produit
- meilleure visibilité du vendeur ("Vendu par")
- pattern d’accueil enrichi (hero, catégories, promotions, nouveautés, réassurance, blog, CTA)
- section blog intégrée : 3 derniers articles + bouton vers le blog
- responsive renforcé sur header, cartes produits et blocs d’accueil

## Compatibilité

- WordPress 6+
- WooCommerce
- WCFM
- Gutenberg

## Structure clé

```text
assets/css/gutenberg-front.css
patterns/home-accueil.html
functions.php
header.php
footer.php
style.css
```
